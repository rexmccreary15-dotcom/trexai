# Install Ollama for Free Unlimited AI

Ollama lets you run AI models **completely free** on your own computer - no API keys, no limits, no costs!

## Step 1: Download Ollama

**Windows Download**: https://ollama.com/download/windows

1. Click the link above
2. Download the installer
3. Run the installer
4. Follow the installation steps

## Step 2: Install a Free AI Model

After installing Ollama, open PowerShell or Command Prompt and run:

```bash
ollama pull llama3.2
```

This downloads a free, fast AI model (about 2GB). It's completely free!

**Other free models you can try:**
- `ollama pull llama3.2` - Fast, good quality (recommended)
- `ollama pull phi3` - Smaller, faster
- `ollama pull mistral` - Good alternative

## Step 3: Start Ollama

Ollama should start automatically after installation. If not:
- Look for "Ollama" in your Start menu
- Click it to start

## Step 4: Test It

Open PowerShell and run:

## Troubleshooting

### "Ollama is not running"
- Check if Ollama is in your system tray (bottom right)
- If not, start it from Start menu
- Or run: `ollama serve` in terminal

### "Model not found"
- Run: `ollama pull llama3.2`
- Wait for download to complete
- Try again

### "Port 11434 already in use"
- Another Ollama instance might be running
- Close other Ollama windows
- Restart Ollama
